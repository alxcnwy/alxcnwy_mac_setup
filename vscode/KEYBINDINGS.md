## VS Code keyboard shortcuts

Canonical keybindings for this setup. These live in `vscode/keybindings.json`, which is symlinked to your VS Code `keybindings.json` so the repo stays the source of truth.

### Panel / terminal

- `cmd+j` — toggle bottom panel
- `cmd+shift+enter` — maximize / restore panel
- `ctrl+\`` — focus terminal
- `cmd+shift+;` — new terminal
- `cmd+shift+k` — kill terminal
- `cmd+shift+\\` — split terminal (when terminal focused)

### Editor / tabs

- `cmd+\\` — split editor
- `cmd+option+right` / `cmd+option+left` — next / previous tab
- `cmd+w` — close active tab
- `cmd+shift+t` — reopen closed tab

### Navigation

- `cmd+shift+p` — command palette
- `cmd+p` — quick open file
- `cmd+shift+f` — search in files
